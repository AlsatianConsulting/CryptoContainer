# Quick Start

## Open An Existing VeraCrypt Container
1. Open the `VeraCrypt` tab.
2. Use `Pick Container` or open a `.hc` file from another app.
3. Enter `Password`.
4. Enter `PIM (optional)` if needed.
5. Add `Keyfiles (optional)` if required.
6. Tap `Open`.
7. Tap `Explore Container` from the `Current Container` card.

## Create A New VeraCrypt Container
1. Open the `VeraCrypt` tab.
2. Tap `Create Volume`.
3. Choose the output file.
4. Pick `Standard` or `Hidden`.
5. Set size, password, optional PIM, file system, algorithm, and hash.
6. Tap `Create`.

## Encrypt One File With AESCrypt
1. Open the `AESCrypt` tab.
2. Tap `Encrypt File`.
3. Pick the input file.
4. Pick the output folder.
5. Enter and confirm the password.
6. Tap `Encrypt`.

## Decrypt One AESCrypt File
1. Open the `AESCrypt` tab.
2. Tap `Decrypt File`.
3. Pick the encrypted file.
4. Pick the output folder.
5. Enter the password.
6. Tap `Decrypt`.
7. Use `Open File` or `Open Folder` after success.

## Share A File Into CryptoContainer
1. Share a file from another app to `CryptoContainer`.
2. Choose one of:
   - `Encrypt Using AESCrypt`
   - `Decrypt Using AESCrypt`
   - `Mount as VeraCrypt Container`
3. If a VeraCrypt container is already open, you can also choose:
   - `Share Into Open VeraCrypt Container`
